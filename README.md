# homebrew-claudecontrol

Homebrew tap for [claudecontrol](https://github.com/grothkopp/claudecontrol) —
a macOS menubar app for the Claude desktop app (unofficial).

```sh
brew tap grothkopp/claudecontrol
brew trust grothkopp/claudecontrol   # newer Homebrew requires trusting third-party taps
brew install claudecontrol
claudecontrol                        # start the menubar app
```

Then grant Accessibility permission: System Settings → Privacy & Security → Accessibility.
